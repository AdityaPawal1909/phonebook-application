# 📇 Contact Manager

A modern and responsive **Contact Manager** web application built using **HTML, CSS, Bootstrap, and JavaScript**. This project allows users to easily manage their contacts with a clean Glassmorphism interface and browser-based Local Storage.

---

## ✨ Features

- 👤 Add New Contact
- ✏️ Edit Existing Contact
- 🗑️ Delete Contact
- 🔍 Search Contacts Instantly
- 🌙 Dark Mode
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- ✨ Modern Glassmorphism UI
- 🎨 Bootstrap 5 Styling
- ⚡ Smooth Hover Animations

---

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

1. Clone this repository:

```bash
git clone https://github.com/AdityaPawal/contact-manager.git
```

2. Open the project folder.

3. Launch `index.html` in your web browser.

No additional installation or dependencies are required.

---

## 📌 Key Highlights

- Store contacts permanently using Local Storage.
- Edit and delete contacts with a single click.
- Search contacts instantly by name or phone number.
- Switch between Light and Dark Mode.
- Responsive layout for desktop, tablet, and mobile devices.
- Attractive Glassmorphism design with Bootstrap components.
